# 📚 مشروع صحيح البخاري الرقمي (Bukhari.TO)

مرجع رقمي متكامل لأصح كتاب بعد كتاب الله عز وجل، تم تطويره ليكون جزءاً من **إمبراطورية سيف الرقمية (Network TO)**.

## 🚀 المميزات التقنية:
- **واجهة مستخدم (UI):** تصميم مينيماليست يدعم القراءة المريحة والوضع المظلم.
- **تعدد الصيغ:** روابط مباشرة لتحميل الكتاب بصيغ (PDF مصور، PDF نصي، ePub، Torrent).
- **التكامل الذكي:** مرتبط بخادم Cloudflare Worker عبر النطاق الفرعي `bukhari.to.com.ly`.
- **الاستضافة:** الكود مستضاف على GitHub Pages والملفات مرتبطة بـ Internet Archive لضمان السرعة والبقاء.

## 🛠️ كيف يعمل؟
هذا المستودع يمثل الواجهة الأمامية (Front-end)، ويتم توجيهه ومعالجته عبر "محرك TO الرئيسي" لضمان تجربة تصفح آمنة وسريعة.
